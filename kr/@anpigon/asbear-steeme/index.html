asbear님의 steeme 소스를 포크하여 수정

작성일 2019-04-19 Edited on 2019-11-11 In 개발 이야기 , 스팀잇(Steemit) Views: Disqus:


Design by @imrahelk

안녕하세요. 안피곤입니다.

@wonsama님이 예전에 작성했던 글을 다시 찾아보려고, @asbear님의 steeme에 오랜만에 접속했습니다. 저는 작성했던 스팀잇 글을 전체적으로 파악하거나 검색 할 때, steeme의 Posting history 기능을 매우 유용하게 사용하고 있습니다. 그런데 조회도 안되고 오류가 발생하네요.



확인해보니 리스팀을 가져오는 함수에서 오류가 발생하고 있었습니다. 그래서 getRebloggedByAsync 함수를 오버라이딩(Overriding) 했습니다.

이것 외에도 API 규격이 조금 바뀐게 있네요.




결국 steeme를 수정하기위해 Github를 포크(Fork)했습니다.

https://anpigon.github.io/steeme/


예전에 활동 하셨던 능력자 분들이 스팀잇에 다시 돌아왔으면 좋겠습니다.


여기까지 읽어주셔서 감사합니다.
